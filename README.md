# ForgeOS

A hobby operating system written completely from scratch.

## Goals

- Learn x86-64 architecture
- Write my own assembler
- Write my own linker
- Write my own compiler
- Write my own bootloader
- Write my own kernel
- Eventually become self-hosting

## Target Hardware

- Samsung XE310XBA-K01US
- Intel Celeron N4000
- 4 GB RAM
- 32 GB eMMC
